# companycam-vibe-check

Hardware logging library for CompanyCam

## Installation

```sh
yarn install @companycam/companycam-vibe-check
```

## Usage

```js
import { multiply } from 'companycam-vibe-check';

// ...

const result = await multiply(3, 7);
```
